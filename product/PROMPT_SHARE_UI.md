# ğŸ¹ VibeVST Prompt Share UI

## Product Vision

A beautiful, minimalist prompt-sharing platform for the VibeVST community. Users can browse, share, and discover VST plugin prompts that others have used to generate audio plugins.

**Design Philosophy:** Apple-inspired glassmorphism with soft edges, subtle animations, and a focus on content over chrome.
